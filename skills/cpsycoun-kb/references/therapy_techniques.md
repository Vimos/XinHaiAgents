# 心理咨询流派技术详解

## 1. 认知行为疗法 (CBT) - 26%

### 核心原理
- **认知三角**: 思维 → 情绪 → 行为
- **核心假设**: 非理性认知导致情绪困扰
- **目标**: 识别并改变负面思维模式

### 核心技术

#### 1.1 认知重构 (Cognitive Restructuring)

**定义**: 识别并改变非理性/扭曲的思维模式

**步骤**:
1. 识别自动思维
2. 检验证据
3. 寻找替代思维
4. 评估情绪变化

**对话示例**:
```
来访者: 我肯定考不上研究生，我太笨了。
咨询师: 你刚才说"我太笨了"，这是你的自动思维。
         我们来检验一下：有什么证据支持你考不上？
         有什么证据反对这个想法？
来访者: 我上次模拟考很差...但我平时成绩还可以。
咨询师: 很好。那如果换个想法："我有能力，但需要改进方法"，
         感觉会怎样？
```

#### 1.2 行为实验 (Behavioral Experiments)

**定义**: 通过实际行动检验负面预测

**应用**: 焦虑、恐惧、回避行为

**对话示例**:
```
来访者: 我不敢在公众场合发言，会出丑。
咨询师: 我们来设计一个行为实验。
         这周在小范围（3-5人）分享你的观点，
         记录实际结果和你的预测有什么不同。
```

#### 1.3 暴露疗法 (Exposure Therapy)

**类型**:
- 系统脱敏（渐进式）
- 洪水疗法（直接面对）

**对话示例**:
```
咨询师: 我们建立一个焦虑等级表，
         从最轻微的情境开始逐步暴露。
         1级：想象演讲场景
         5级：在小组中发言
         10级：在大会上演讲
```

#### 1.4 家庭作业 (Homework)

**常见任务**:
- 思维记录表
- 行为激活计划
- 放松训练

## 2. 精神分析疗法 - 16%

### 核心原理
- **潜意识冲突**: 早期经历影响当前行为
- **防御机制**: 自我保护的无意识策略
- **移情与反移情**: 咨询关系中的动态

### 核心技术

#### 2.1 自由联想 (Free Association)

**定义**: 不加筛选地说出任何想法

**对话示例**:
```
咨询师: 请放松，说出你脑海中出现的任何想法，
         不要审查，想到什么说什么。
来访者: 工作...压力...父亲...小时候...
咨询师: 你提到了父亲，能多说一些吗？
```

#### 2.2 梦的解析 (Dream Analysis)

**定义**: 分析梦境中的潜意识内容

**对话示例**:
```
来访者: 我梦见被困在黑暗的房间里。
咨询师: 这个梦可能反映了你现实中的被困感。
         梦中的黑暗让你联想到什么？
```

#### 2.3 移情分析 (Transference Analysis)

**定义**: 分析来访者对咨询师的感情投射

**对话示例**:
```
来访者: 我觉得你不会理解我，就像我父亲一样。
咨询师: 你把我体验成了你的父亲。
         这种感觉在你的生活中常见吗？
```

## 3. 人本主义疗法 - 5%

### 核心原理
- **自我实现**: 人有成长的内在潜能
- **无条件积极关注**: 接纳而非评判
- **真诚一致**: 咨询师的真实存在

### 核心技术

#### 3.1 积极倾听 (Active Listening)

**要素**:
- 全神贯注
- 反映感受
- 澄清内容

**对话示例**:
```
来访者: 我不知道该怎么办，感觉很迷茫。
咨询师: [点头，眼神接触]
         你感到迷茫，面对选择时不知何去何从。
来访者: 是的，我不知道哪个选择是对的。
咨询师: 你希望能做出正确的选择，
         这种不确定性让你感到焦虑。
```

#### 3.2 无条件积极关注

**定义**: 无条件接纳来访者，不评判

**对话示例**:
```
来访者: 我做了一件很糟糕的事...
咨询师: 无论发生了什么，我都在这里倾听。
         你愿意分享吗？
```

## 4. 家庭治疗 - 12%

### 核心原理
- **系统观点**: 个体问题反映家庭系统
- **循环因果**: 互动模式维持问题
- **家庭结构**: 边界、层级、联盟

### 核心技术

#### 4.1 家谱图 (Genogram)

**用途**: 了解家庭历史和多代模式

**对话示例**:
```
咨询师: 我们一起画一个家谱图，
         包括三代人的关系和重要事件。
         这能帮助我们理解问题的家庭背景。
```

#### 4.2 系统提问

**类型**:
- 循环提问："当你生气时，你妈妈会怎么做？"
- 未来提问："如果问题解决了，家庭会有什么不同？"
- 例外提问："有没有什么时候问题没有发生？"

## 5. 后现代疗法 - 13%

### 5.1 叙事疗法 (Narrative Therapy)

**核心**: 问题不等于人，重写生命故事

**技术**: 外化问题、寻找独特结果

**对话示例**:
```
来访者: 我很抑郁。
咨询师: 抑郁是什么时候开始影响你的生活的？
来访者: 大概半年前...
咨询师: 在抑郁到来之前，你是一个怎样的人？
```

### 5.2 焦点解决短期治疗 (SFBT)

**核心**: 关注解决方案而非问题

**技术**: 奇迹提问、量尺提问、例外询问

**对话示例**:
```
咨询师: 假设今晚睡觉时发生了一个奇迹，
         你的问题解决了。
         明天醒来，你会发现有什么不同？
```

## 流派选择指南

| 问题类型 | 推荐流派 | 理由 |
|---------|---------|------|
| 焦虑障碍 | CBT | 证据充分，短程高效 |
| 抑郁症 | CBT/人本 | 认知重构 + 情感支持 |
| 人格障碍 | 精神分析 | 深层人格改变 |
| 家庭冲突 | 家庭治疗 | 系统视角 |
| 自我探索 | 人本/后现代 | 个人成长导向 |
| 创伤后应激 | CBT/精神分析 | 处理创伤记忆 |

## 整合应用

```python
    def __init__(self):
        self.techniques = {
            "CBT": ["cognitive_restructuring", "behavioral_experiment"],
            "Humanistic": ["active_listening", "unconditional_positive_regard"],
            "Psychodynamic": ["free_association", "transference_analysis"]
        }
    
    def select_techniques(self, problem_type, client_preference):
        """根据问题和偏好选择技术"""
        if problem_type == "anxiety":
            return self.techniques["CBT"] + ["active_listening"]
        elif problem_type == "self_exploration":
            return self.techniques["Humanistic"] + ["free_association"]
        # ...
```
